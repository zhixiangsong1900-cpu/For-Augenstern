=====================================
For Augenstern - 资源文件夹
=====================================

此文件夹用于存放你要替换的资源文件。

📁 文件夹结构：
├── photos/          放入你们的照片（10张）
│   ├── photo1.jpg
│   ├── photo2.jpg
│   └── ... (photo1-photo10)
│
├── audio/           放入背景音乐（可选）
│   ├── bgm1.mp3     第一首BGM
│   ├── bgm2.mp3     第二首BGM
│   └── bgm3.mp3     第三首BGM
│
└── bg/              背景图片（可选）
    └── ...


📷 照片说明：
- 建议尺寸：正方形 400x400 或 600x600
- 格式：JPG/PNG
- 文件名：photo1.jpg 到 photo10.jpg
- 在 js/config.js 的 ORNAMENTS 中修改对应的名称和消息

🎵 音乐说明：
- 格式：MP3
- 在 js/config.js 的 AUDIO.BGM_LIST 中修改曲目名称

💡 提示：
- 如果没有放置图片，游戏会使用emoji作为占位符
- 如果没有放置音乐，游戏会使用程序生成的音效
- 游戏支持 file:// 协议直接运行

🚀 运行方式：
1. 直接双击 index.html 在浏览器中打开
2. 或使用 Live Server 等本地服务器
3. 或部署到 GitHub Pages

❤️ 祝她圣诞快乐！
